全栈开发

何为全栈

全栈工程师，英文 Full Stack developer。是指掌握多种技能，并能利用多种技能独立完成产品的人。
技术有两个发展方向，一种是纵向一种是横向的，横向的是瑞士军刀，纵向的是削铁如泥的干将莫邪。这两个方向都没有对与错，发展到一定程度都会相互溶合，就好比中国佛家禅修的南顿北渐，其实到了最后，渐悟与顿悟是一样的，顿由渐中来。

全栈架构

技术 描述
git 代码版本管理
koa node 框架
react 前端框架
mongodb nosql 数据库
mongoose mongodb 模型
jenkins 自动化工具
pm2 部署 node 项目
docker 容器
portainer docker 可视化管理

oss,cos,七牛用来存储对象文件

java，python 全部用来提供 api

工作流程

IDE 编写代码，git 提交到 github，触发 webhooks，jenkins 自动打包，打包完成触发钉钉机器人发消息。node 项目使用 pm2 启动或者 docker，react 项目打包到 nginx 下或者 docker 里。使用 portainer 查看 docker 容器的状态。

开发只需要安安静静的编写业务逻辑，全自动部署，美滋滋啊。
